# Housing Price Prediction 🏠

Machine Learning project for predicting housing prices using Python and Scikit-learn.

## Features

* Data preprocessing
* Handling missing values
* Feature scaling
* Pipeline creation
* Model training
* Model evaluation using RMSE and MAE
* Housing price prediction

## Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib

## Files

* `main.py` → source code
* `housing.csv` → dataset
* `model.pkl` → trained model
* `pipeline.pkl` → preprocessing pipeline
